# MoveToDownload

A lightweight Firefox add-on that downloads images automatically when dragging and releasing them

---

## Features

- Click, drag and drop an image to begin downloading
- Configurable folder path setting for downloading to (relative to currently set Downloads path)
- Images timestamping according to the moment they were downloaded

---

## Installation

1. Clone this repostiory

2. Open Firefox and navigate to `about:debugging`

3. Click **This Firefox** on the left menu

4. Click **Load Temporary Add-on..** and select `manifest.json` in the cloned repostiory
